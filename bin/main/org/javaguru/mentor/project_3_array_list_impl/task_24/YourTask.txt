В виде опциональных заданий (для получения практики),
можете поработать над реализацией следующих методов:
- boolean contains(Object o);
- void remove(Object o);
- void removeAll(Object o);
- void add(int index, Object o);
- clear();
- indexOf(Object o);
- addFirst(Object o);
- addLast(Object o);

Работу над каждым методом постройте по следующему плану:
- составьте описание и опишите требования к методу в письменной форме;
- составьте тест кейсы для тестирования;
- реализуйте метод;
- напишите юнит тесты.

Этот подход поможет вам научиться фиксировать требования к
программному коду до начала реализации.

Вы научитесь записывать требования в письменной форме, это очень важный навык.

На основе требований вы можете подготовить тест кейсы для тестирования,
еще до начала реализации! Test First Development в действии.

Все это вместе в конечном счете приведет к созданию вами более
продуманного и качественного кода!
