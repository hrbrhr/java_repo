Перед тем как приступить к работе над реализацией нашего списка
давайте поговорим о массивах и их ограничениях.

Массивы в Java это классные конструкции позволяющие хранить
в оперативной памяти большое количество данных. Например:
создайте массив на 1000 ячеек и сохраните в него 1000 целых чисел.
Или создайте массив на миллион ячеек и сохраните в него ссылки
на объекты класса Book (книга, например можно использовать для
представления электронных книг). Массив позволяет хранить много
данных, но!

Какое самое большое ограничение есть у массивов в Java?

Правильно, размер массива после его создания менять нельзя.

То есть на момент создания массива заранее нужно знать его размер.
Получается, что перед тем как вы будете в программе использовать
массив, вы уже должны знать его размер. После создания размер
массива изменить уже нельзя.
