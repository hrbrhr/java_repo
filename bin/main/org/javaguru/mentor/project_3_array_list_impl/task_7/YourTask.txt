Для того чтобы наша будущая реализация получилась качественной,
нам нужно подготовить так называемую "техническую документацию"
с более подробным описанием поведения каждого из ниже перечисленных
методов:

- add(element) - добавить элемент в список
- remove(elementIndex) - удалить элемент из списка под указанным индексом
- get(elementIndex) - достать элемент из списка по индексу
- size() - вернуть размер списка
- isEmpty() - вернуть пустой ли список или нет

Техническая документация должна описывать поведение метода
и различные corner cases (пограничные случаи). В документацию
можно добавить примеры!
