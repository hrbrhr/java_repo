Техническая документация для метода:
- add(element) - добавить элемент в список

Метод add(element) должен добавлять новый элемент в список.

Новый элемент должен добавляться в конец списка.

После добавления в список новый элемент становится последним в списке.

После выполнения операции add(element) длина списка должна увеличиться на 1.

Список должен позволять добавлять одинаковые (эквивалентные) объекты
сколько угодно раз, операция add(element) не должна проверять есть ли
такой элемент в списке или его нет, она просто должна добавлять новый
элемент в список.

PS: для простоты разработки будем считать, что в списке мы будем
хранить целые числа типа int.

В последних задачах проекта мы переделаем наш список на хранение
объектов любого класса. Для этого воспользуемся генериками, но они
не цель нашего, цель этого проекта изучить внутреннюю механику
работы списка построенного на массивах.
