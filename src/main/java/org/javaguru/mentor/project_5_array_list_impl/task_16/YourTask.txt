Напишите первые юнит тесты на методы класса IntegerArrayListImpl.

Напишите первые тесты на методы:
    int size();
    boolean isEmpty();

Тесты должны проверять следующие требования:
- размер списка после создания должен быть 0.
- метод isEmpty() сразу после создания должен возвращать true.
- когда метода size() возвращает 0, isEmpty() должен возвращать true.
