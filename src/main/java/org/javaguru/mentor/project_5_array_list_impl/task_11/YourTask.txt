Техническая документация для метода:
- size() - вернуть размер списка

Метод должен возвращать текущую длину списка.
Длина списка равна количеству элементов на данный
момент находящихся в списке. Результат возвращается
как число типа int. Если в списке нет элементов
(пустой список), то метод должен вернуть 0.

- isEmpty() - вернуть пустой ли список или нет
