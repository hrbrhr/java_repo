Техническая документация для метода:
- remove(elementIndex) - удалить элемент из списка под указанным индексом

Метод remove(elementIndex) должен удалять элемент из списка,
который находиться в нем под индексом elementIndex.

Если elementIndex < 0 то метод должен кидать ошибку ArrayIndexOutOfBoundException.

Если elementIndex >= длине списка, то метод должен кидать ошибку ArrayIndexOutOfBoundException.

После удаления элемента из списка длина списка должна уменьшаться на 1.
