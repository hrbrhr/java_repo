Реализуйте метод remove(elementIndex).

Напоминаем, что метод должен:
- удалять элемент из списка по указанному индексу.
- возвращать ошибку если указанный индекс выходит за границы списка.

Напишите реализацию метода remove(elementIndex).

Составьте тест кейсы для метода remove(elementIndex), которые
