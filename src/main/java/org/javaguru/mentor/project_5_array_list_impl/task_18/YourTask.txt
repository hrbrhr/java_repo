Добавьте юнит тесты для метода add(newElement).

Напоминаем, что метод должен:
- создать массив на одну ячейку большего размера;
- скопировать все элементы из старого массива в новый на свои позиции;
- самым последним добавить новый элемент;
- заменить старый массив на новый.

Давайте составим набор тест кейсов, которыми мы воспользуемся
при написании тестов:
- создаем список, добавляем один элемент, длина должна стать 1;
- создаем список, добавляем один элемент, isEmpty() = false;
- создаем список, добавляем два разных элемента, длина должна стать 2;
- создаем список, добавляем два одинаковых элемента, длина должна стать 2;
