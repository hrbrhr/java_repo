В прошлом задании мы просили вас подготовить список действий
(методов), который по вашему мнению должен уметь выполнять список.

Давайте сравним результаты:

- add(element) - добавить элемент в список
- remove(elementIndex) - удалить элемент из списка под указанным индексом
- get(elementIndex) - достать элемент из списка по индексу
- size() - вернуть размер списка
- isEmpty() - вернуть пустой ли список или нет

У вас мог получиться немного другой список, это нормально.
Это далеко не полный набор методов, которые могут быть у списка.
