Техническая документация для метода:
- get(elementIndex) - достать элемент из списка по индексу

Метод должен возвращать элемент из списка под указанным индексом.

Если elementIndex < 0 то метод должен кидать ошибку ArrayIndexOutOfBoundException.

Если elementIndex >= длине списка, то метод должен кидать ошибку ArrayIndexOutOfBoundException.
